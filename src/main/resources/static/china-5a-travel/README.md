# 中国5A旅游景区网站

## 目录说明

- `index.html`：页面入口
- `styles.css`：样式文件
- `app.js`：搜索、筛选、排序、分页与详情展开逻辑
- `data/attractions.json`：景区数据（含附近酒店与餐饮）

## 数据覆盖

- 覆盖范围：全国 5A 旅游景区
- 省级行政区覆盖：31
- 当前景区数量：360（以数据源解析结果为准）
- 每个景区包含：
  - 位置（省/市/区县）
  - 门票参考价格
  - 附近酒店（名称、位置、距离、价格、评分、电话）
  - 附近餐饮（名称、位置、距离、人均、类型、营业时间、评分、电话）

## 访问方式

Spring Boot 启动后访问：

`/china-5a-travel/`

## 说明

酒店与餐饮为行程规划参考信息，请以实时预订平台和景区官方公告为准。
